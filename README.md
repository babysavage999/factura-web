# 🧾 Factura Web

Sistema de facturación e inventario diseñado para PyMEs colombianas. Permite gestionar clientes, productos y generar facturas de forma simple y ordenada.

## ✨ Características

- 📋 Gestión de clientes
- 📦 Gestión de productos e inventario
- 🧾 Generación de facturas
- 🔐 Autenticación de usuarios
- 📊 Dashboard con resumen de facturación

## 🛠 Stack tecnológico

- **Framework:** Next.js 14
- **UI:** Tailwind CSS + Radix UI
- **Lenguaje:** TypeScript
- **Gráficas:** Recharts

## 🚀 Cómo correr el proyecto localmente

```bash
# Clona el repositorio
git clone https://github.com/babysavage999/factura-web.git

# Entra a la carpeta del proyecto
cd factura-web/billing.app.cloud

# Instala dependencias
npm install

# Corre el servidor de desarrollo
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.
